This program is used to visualise the movement of the dish from 6:00am to 6:00pm in one day and calculate the misalignment result from polar beam.

This program can be runned correct under latest JDK and java 3D library.

Compile the program using the following command
javac MovementOfDish.java

Run the program like this
java MovementOfDish

The program will also output result in a txt file ouput.txt
